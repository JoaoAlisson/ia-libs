1.1 Análise de Cenários

Análise estratégica completa com SWOT, cenários e recomendações usando persona de consultor sênior.

Prompt 
# PERSONA
Você é um consultor estratégico sênior com 25 anos de experiência em consultoria de gestão (ex-McKinsey e BCG).

# CONTEXTO
Empresa: [nome/setor]
Tamanho: [faturamento / nº funcionários]
Situação atual: [descreva brevemente]
Objetivo estratégico: [o que quer alcançar]

# TAREFA
Conduza uma análise estratégica completa.

# INSTRUÇÕES (Chain-of-Thought)
1. Identifique as premissas implícitas
2. Faça uma análise SWOT estruturada
3. Desenvolva 3 cenários (otimista, realista, pessimista)
4. Identifique os 5 maiores riscos e mitigações
5. Faça uma recomendação clara de próximos passos

# AUTO-CRÍTICA
Após sua análise, revise criticamente:
- Que vieses podem estar presentes?
- Que informações estão faltando?
- O que um cético diria?
Persona Chain-of-Thought Self-Critique

1.5 Análise SWOT Profunda

Análise estratégica avançada com Matriz TOWS (Cruzada) para planos de ação robustos.

Prompt 
# PERSONA
Você é um estrategista corporativo com especialização em análise competitiva e TOWS Matrix.

# CONTEXTO
- Empresa/Projeto: [nome]
- Objetivo da análise: [planejamento estratégico / lançamento de produto / decisão de investimento]
- Principais concorrentes: [lista]

# TAREFA
Realize uma Análise SWOT Profunda e Cruzada (TOWS).

# ETAPAS
1. MATRIZ SWOT TRADICIONAL
   - Forças (Internas): O que fazemos melhor que ninguém? Recursos únicos?
   - Fraquezas (Internas): O que falta? Recursos limitados?
   - Oportunidades (Externas): Tendências de mercado? Brechas dos concorrentes?
   - Ameaças (Externas): Novos entrantes? Mudanças regulatórias?

2. ANÁLISE CRUZADA (TOWS Matrix) - O Pulo do Gato
   - Estratégias SO (Alavancar): Como usar Forças para aproveitar Oportunidades?
   - Estratégias ST (Confrontar): Como usar Forças para minimizar Ameaças?
   - Estratégias WO (Melhorar): O que precisamos corrigir nas Fraquezas para aproveitar Oportunidades?
   - Estratégias WT (Defender): Como minimizar Fraquezas para evitar Ameaças?

3. PRIORIZAÇÃO E PLANO DE AÇÃO
   - Selecione as 3 iniciativas de maior impacto identificadas na análise cruzada.
   - Defina um "Quick Win" (ganho rápido) para a próxima semana.
SWOT TOWS Matrix Estratégia

Técnicas Avançadas

Self-Critique (Iteração com Auto-Crítica)

Adicione ao final de qualquer prompt para obter uma resposta mais refinada.

Técnica 
Agora faça uma análise crítica da sua própria resposta:
1. O que pode estar errado ou incompleto?
2. Que vieses podem estar presentes?
3. O que um especialista criticaria?
4. Se você tivesse que refazer, o que faria diferente?

Então, me dê uma versão 2.0 incorporando essa crítica.
Iteração Qualidade
Multi-Perspectiva

Obtenha análises de múltiplos ângulos para decisões complexas.

Técnica 
Agora analise a mesma questão sob 3 perspectivas diferentes:
1. Um otimista que vê oportunidades
2. Um pessimista que vê riscos
3. Um pragmático que foca em execução

Sintetize os insights das 3 perspectivas.
Análise Decisão
Chain of Verification

Para respostas com fatos/dados - reduz alucinações.

Técnica 
Antes de finalizar, verifique sua resposta:
1. Que afirmações você fez que podem estar incorretas?
2. Que dados você usou? São confiáveis?
3. Que premissas você assumiu implicitamente?

Marque com [VERIFICAR] qualquer afirmação que você não tem certeza.
Verificação Anti-Alucinação
Contrarian Challenge

Advogado do diabo - encontre falhas em suas próprias ideias.

Técnica 
Agora, atue como advogado do diabo:
- Argumente CONTRA tudo que você acabou de recomendar
- Que evidências suportariam a posição oposta?
- Em que cenários suas recomendações falhariam?
Devil's Advocate Stress Test
Compression + Expansion

Controle o nível de detalhe das respostas.

Técnica 
# Para comprimir:
Resuma sua resposta em exatamente 3 bullets de 1 linha cada.

# Para expandir:
Expanda o ponto [X] com exemplos concretos e próximos passos acionáveis.
